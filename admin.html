🚀 How to Deploy

1. Go to vercel.com → sign up free

2. Upload index.html + admin.html to GitHub

3. Vercel → New Project → import from GitHub → Deploy

4. Site live at yoursite.vercel.app in 5 minutes

5. Bookmark yoursite.vercel.app/admin.html — your private door
